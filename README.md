The issue stems from improper input handling that enables a cross-site scripting–style condition, allowing a remote attacker to disclose sensitive information over the network. Exploitation may occur without user interaction and with low attack complexity, potentially resulting in data exfiltration via automated features such as Copilot-driven network requests. At the time of disclosure, no public proof-of-concept exploit code or active exploitation in the wild had been reported, and Microsoft assessed the likelihood of exploitation as low, though an official patch has been released and organizations are advised to apply updates promptly.
