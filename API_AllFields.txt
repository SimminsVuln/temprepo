On December 12th Apple issued a security advisory for CVE-2025-43529, a WebKit memory corruption vulnerability that may lead to arbitrary code execution when processing maliciously crafted web content. Apple has confirmed that this issue may have been exploited in an extremely sophisticated, targeted attack against specific individuals on versions of iOS prior to iOS 26. Due to active exploitation and the potential for full compromise through web content, this vulnerability requires immediate patching across all affected Apple devices and platforms.

On December 12th Apple issued a security advisory for CVE-2025-14174, a WebKit use-after-free vulnerability that can result in arbitrary code execution when handling malicious web content. Apple and Google Threat Analysis Group have indicated this issue has been exploited in highly targeted attacks against select users. Given the confirmed real-world exploitation and the severity of impact, organizations should prioritize patch deployment and ensure all supported Apple operating systems are updated without delay.
